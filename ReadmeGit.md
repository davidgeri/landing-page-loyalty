## Daftar Commit Message
 
| Type | Kapan Dipakai |
|---|---|
| `feat` | Nambah fitur baru |
| `fix` | Benerin bug |
| `chore` | Seperti install package atau update package, config, dll bukan fitur/bugfix |
| `docs` | Ubah dokumentasi README, komentar kode |
| `refactor` | Ubah struktur kode tanpa nambah fitur atau benerin bug |

# Example 

```bash
    git commit -m "feat: add new feature humberger"
    git commit -m "fix: priview data time line"
    git commit -m "chore: update tailwind version 4.x"
    git commit -m "docs: change readme"
    git commit -m "refactor: simplify state management in dashbaord"

```
